# Privacy Policy – EUDR DAO Digital Twin Engineer (DTE)

The EUDR DAO Digital Twin Engineer (DTE) is a public, read-only governance assistant.

## Data Usage
- No personal data is collected, stored, or processed by this GPT.
- The GPT does not execute code or perform automated decisions.
- All information used is derived from publicly available documentation and repositories.

## User Interactions
- Conversations occur entirely within the ChatGPT platform and are governed by OpenAI’s privacy policy.
- This project does not receive, log, or store user inputs.
